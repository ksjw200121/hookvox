# Analyze Prompt 改動建議（偏 IG + 法規合規）

依你提供的陪跑班／金融專班 PDF 統整，並加上「各行各業法規、金融招攬」的合規要求。**以下僅為建議，不會直接改程式，你確認後我再改。**

---

## 一、從 PDF 統整出的重點（給 prompt 用）

### 1. IG 專屬（與 TikTok / YT Shorts 區隔）

- **Reels 演算法**：出現在首頁「探索」、推薦欄位；要留粉需「版面主題一致」+「資訊價值」。
- **完播／互動參考**：完播率盡量 15–30 秒內；按讚約 5%–10%；留言／分享約觀看 0.1%；珍藏約 0.2%。
- **內容分工**：貼文／Reels 做「主要定位內容」給陌生人；限動做生活、信任、話題。分析時可標註「適合 Reels 還是貼文」。
- **IG 畫面偏好**（課程反覆提到）：乾淨背景、留白、主視覺清楚、字體一致；避免花色盤子、雜亂、畫面太滿。分析可多一句「以 IG 版面／視覺」的建議。
- **開口要大、主題一致、資訊落差**（尤其金錢／省錢／補助類）：用廣度被看見、用深度經營粉絲；貼上標籤後要往同主題深度經營。

### 2. 爆款本質（與你現有 prompt 對齊）

- 完播→選題對；按讚→共鳴共情；留言→話題感參與感；收藏→有用；分享→你說出我想說的。
- 正確順序：**事件／衝突／錯誤 → 才輪到專業**。不能只有專業，演算法吃：衝突、認知反差、立場、情緒。
- 好開場：直接給結果、點痛點、講代價（不做會怎樣）。壞開場：「今天來分享」「我來教你」「最近很多人問」。

### 3. 金融／專業類（來自金融專班 PDF）

- 陌生人不會因「保險怎麼配置、ETF 怎麼選、利率怎麼算」停下來；要從**情境**切入（例如：為什麼年薪破百帳戶還是空的、房貸利率差 0.3% 的影響）。
- 關鍵字：年薪/所得稅、勞保勞退、補助、繼承、贈與、發票、責任制、特休等「觀眾會搜的」；避免只寫給同行看的術語。
- **不建議生成**：承諾報酬、保證獲利、貶低同業、未經核實的數據、誘使不當招攬的文案（見下方法規）。

### 4. 法規與合規（你要的「不能違法、被檢舉、罰錢、金融業被停止招攬」）

- **通用**：不生成違反法律、具誤導性、可被檢舉或導致民事/行政責任的內容；不鼓勵抄襲（可參考多來源、消化後再輸出）。
- **金融業**：不生成可能導致「保險/證券/銀行招攬被處分」的內容，例如：  
  - 保證獲利、未經核准的投資建議、不當比較商品、誤導性數字、未加警語的報酬描述。  
  - 若內容涉及投資/保險/理財，分析或延伸腳本應提醒「僅供參考、不構成投資/投保建議，須依自身狀況並遵守當地法規」。
- **其他行業**：醫療、法律、食品等若有廣告或宣稱，也需符合該行業法規；分析時可標註「若涉及專業/商業宣稱，請創作者自行確認合規」。

## 二、我建議的改動（請你逐項確認）

### A. 開頭與角色（更偏 IG）

- **改**：把「短影音爆款內容分析師，熟悉台灣 IG Reels、TikTok、YouTube Shorts」改成  
  **「以 IG Reels 為主的短影音爆款內容分析師，熟悉台灣 IG 演算法與 Reels 推薦邏輯，兼顧 TikTok、YouTube Shorts 的共通原則。」**
- **理由**：明確以 IG 為主，分析結果會更貼 IG 的留粉、版面、探索頁邏輯。

### B. 新增「IG 專屬」一小段

- **加**：  
  - Reels 出現在探索與推薦欄位，要留粉需「主題一致」+「資訊價值」；分析時可註明此片較適合 Reels 或貼文。  
  - IG 畫面偏好：乾淨、留白、主視覺清楚；避免雜亂、太滿、字體不一。  
  - 完播／互動參考：完播 15–30 秒、按讚約 5%–10%、留言/分享約 0.1%、珍藏約 0.2%（僅供解讀爆款用）。
- **理由**：讓分析結果自帶「IG 視角」，而不是泛短影音。

### C. 新增「法規與合規」強制段落

- **加**：  
  - **【法規與合規】**  
    - 不臆測或生成違反法律、具誤導性、易被檢舉或導致民事/行政責任的內容。  
    - **金融/投資/保險類**：不生成保證獲利、未經核准的投資建議、不當比較、誤導性數字；若內容涉及此類，須在分析或建議中提醒「僅供參考，不構成投資/投保建議，創作者須自行確認合規並遵守當地法規（含金融招攬規定）」。  
    - **其他行業**：若涉及醫療、法律、食品等專業或商業宣稱，分析可註明「創作者須自行確認符合該行業法規」。
- **理由**：從系統提示就限制輸出，降低違法、被檢舉、金融招攬處分風險。

### D. 現有「爆款本質」「好開頭 vs 壞開頭」微調

- **改**：在「壞開頭」範例中補一句課程裡常提到的：  
  「最近很多人問」「我來教你」「今天跟你分享」等，並保留你原有的「今天來分享一個知識給大家」。  
- **可選**：在 keyFormula / contentStructure 的說明裡加一句「若為金融/專業類，從情境、痛點、代價切入，而非從術語或產品開始」。
- **理由**：與 PDF 的「開場健檢」「情境導向」一致。

### E. JSON 輸出是否加欄位（可選）

- **選項 1**：不新增欄位，只在 system 裡加 IG + 法規說明（改動最小）。  
- **選項 2**：新增一個選填欄位，例如 `"igNote": "此片較適合 Reels／貼文；版面建議…"` 或 `"complianceNote": "涉及金融時請加警語…"`。  
- **建議**：先做選項 1，若你之後需要結構化「合規提醒」再加欄位。

## 三、改之前 vs 改之後 差異一覽

| 項目 | 改之前 | 改之後 |
|------|--------|--------|
| **平台重心** | 台灣 IG Reels、TikTok、YT Shorts 並列 | **以 IG Reels 為主**，熟悉 IG 演算法與推薦邏輯，TikTok/YT 為共通原則 |
| **IG 專屬** | 無 | 有 Reels/貼文定位、版面與畫面偏好、完播與互動參考比例 |
| **法規／合規** | 無 | **有【法規與合規】段落**：不違法、不誤導、不導致招攬處分；金融類提醒警語與自行合規；他行業提醒自行確認法規 |
| **開場壞範例** | 僅「今天來分享一個知識給大家」 | 補上「最近很多人問」「我來教你」「今天跟你分享」等 |
| **金融/專業類** | 未特別提醒 | 可選：keyFormula/結構說明加「從情境、痛點切入，勿從術語開始」；分析若涉金融可帶出合規提醒 |
| **輸出欄位** | 維持現有 JSON | 維持不變（或你同意後再加 igNote / complianceNote） |
