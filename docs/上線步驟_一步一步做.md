# Hookvox 上線 — 一步一步做（詳細版）

> 完全照順序做，做完一步再打勾。卡住時把「你現在卡在第幾步、畫面長怎樣」傳給我就好。

---

## 第一部分：把程式碼放到 GitHub（這樣 Vercel 才抓得到）

### 步驟 1：你有沒有 GitHub 帳號？

- **有** → 下一步（步驟 2）
- **沒有** → 打開 https://github.com → 點右上角 **Sign up** → 用 Email 註冊一個帳號 → 再回來做步驟 2

---

### 步驟 2：在 GitHub 建立一個「倉庫」

1. 登入 GitHub 後，點左上角 **綠色按鈕「New」**（或網址打 https://github.com/new）
2. **Repository name** 那一格填：`hookvox-ai`（或你喜歡的名字，不要有空格）
3. 下面 **Public** 選好，其他都不用改
4. 最下面 **不要** 勾選 "Add a README file"
5. 點 **Create repository**
6. 建立好後會看到一個頁面，上面有一排網址，先**不要關掉**，等一下會用到

---

### 步驟 3：用電腦上的「命令列」把專案傳上 GitHub

1. 打開你放 Hookvox 專案的資料夾（例如 `Desktop\viralscript-ai`）
2. **在該資料夾裡**打開「命令提示字元」或「PowerShell」：
   - 方法 A：在資料夾空白處按 **Shift + 右鍵** → 選「**在此處開啟 PowerShell 視窗**」
   - 方法 B：檔案總管網址列輸入 `cmd` 按 Enter
3. 在 PowerShell 黑色視窗裡**不要輸入「bash」**，只輸入下面每一行指令（一次一行，貼上後按 Enter）：

第一行，貼上後按 Enter：
```
git init
```
（若出現「already exists」沒關係，繼續）

第二行：
```
git add .
```

第三行：
```
git commit -m "first upload"
```

4. 下一行要填**你的 GitHub 倉庫網址**。請到步驟 2 那個 GitHub 頁面，點綠色 **Code** 按鈕，選 **HTTPS**，複製網址（長得像 `https://github.com/你的帳號/hookvox-ai.git`），把下面**整行**裡的 `你的倉庫網址` 換成你複製的網址後，貼到 PowerShell 按 Enter：

```
git remote add origin 你的倉庫網址
```
例如你複製的是 `https://github.com/abc123/hookvox-ai.git`，就貼：`git remote add origin https://github.com/abc123/hookvox-ai.git`

5. 再貼這兩行（一次一行）：

```
git branch -M main
```
按 Enter 後再貼：
```
git push -u origin main
```

6. 可能會要你輸入 **GitHub 帳號** 和 **密碼**（若用密碼登入，現在 GitHub 多半會要你用 **Personal access token** 當密碼，可到 GitHub → Settings → Developer settings → Personal access tokens 建立一個，權限勾 repo）

7. 成功的話會看到 `main -> main` 之類的訊息，GitHub 網頁重新整理會看到你的檔案

**若這一步失敗**：把「你貼了哪幾行、視窗出現什麼錯誤訊息」複製給我（或截圖），我幫你對應。

## 第二部分：用 Vercel 部署（讓網站有網址）

### 步驟 4：註冊 Vercel

1. 打開瀏覽器，網址列輸入：**https://vercel.com**
2. 點右上角 **Sign Up**
3. 選 **Continue with GitHub**（用 GitHub 登入）
4. 若跳出授權，點 **Authorize** 允許 Vercel 讀你的 GitHub
5. 登入後會進到 Vercel 的畫面（可能問你團隊名稱，隨便填一個即可）

---

### 步驟 5：在 Vercel 匯入你的專案

1. 在 Vercel 畫面點 **Add New…**（或 **Create**）→ 選 **Project**
2. 會看到 **Import Git Repository**，下面列出你的 GitHub 專案
3. 找到 **hookvox-ai**（或你步驟 2 取的名字），右邊點 **Import**
4. 下一頁 **不要馬上點 Deploy**，先看下面「Environment Variables」

---

### 步驟 6：在 Vercel 填「環境變數」（最重要）

這一頁會有一個區塊叫 **Environment Variables**。你要把本機 `.env.local` 裡的東西一筆一筆加進來。

**怎麼加一筆：**

1. **Key** 那一格：填變數名稱（例如 `NEXT_PUBLIC_SUPABASE_URL`），一個字都不能錯
2. **Value** 那一格：貼上對應的值（從你電腦的 `.env.local` 複製）
3. 右邊 **Environment** 選 **Production**
4. 點 **Add** 或 **Add Another**

**請照下面清單，一筆一筆加（名稱必須完全一樣）：**

| 序號 | Key（變數名稱） | Value 從哪裡來 |
|------|------------------|----------------|
| 1 | NEXT_PUBLIC_SUPABASE_URL | 你 .env.local 裡同一行的值（等號右邊，引號可去掉） |
| 2 | NEXT_PUBLIC_SUPABASE_ANON_KEY | 同上 |
| 3 | SUPABASE_SERVICE_ROLE_KEY | 同上 |
| 4 | DATABASE_URL | 同上 |
| 5 | OPENAI_API_KEY | 同上 |
| 6 | ANTHROPIC_API_KEY | 同上 |
| 7 | ECPAY_MERCHANT_ID | 同上 |
| 8 | ECPAY_HASH_KEY | 同上 |
| 9 | ECPAY_HASH_IV | 同上 |
| 10 | NEXT_PUBLIC_APP_URL | **先填** `https://你的專案名.vercel.app`（專案名在 Vercel 匯入時會顯示，例如 hookvox-ai，就填 `https://hookvox-ai.vercel.app`；**不確定可先填** `https://hookvox-ai.vercel.app`，部署完再改） |
| 11 | ECPAY_NOTIFY_URL | 填 `https://你的專案名.vercel.app/api/ecpay/notify`（和上面同一個網址，後面加 `/api/ecpay/notify`） |
| 12 | ECPAY_RETURN_URL | 填 `https://你的專案名.vercel.app/payment/success` |
| 13 | NEXTAUTH_SECRET | .env.local 裡有就填，沒有就填一串亂碼（例如 32 個英數字） |
| 14 | NEXT_PUBLIC_TURNSTILE_SITE_KEY | .env.local 裡同一行的值 |
| 15 | ADMIN_SUPABASE_IDS | .env.local 裡有就填，沒有可先不填 |

**若你願意**：也可以把 `.env.local` 的**內容**（把密碼、API Key 打馬賽克或換成 xxx）貼給我，我幫你列成「Key = Value」清單，你照貼到 Vercel 就好。

加完後，點下方 **Deploy**。

---

### 步驟 7：等部署跑完

1. 畫面會顯示 **Building…**，等 1～3 分鐘
2. 完成後會出現 **Congratulations** 和一個網址，例如：`https://hookvox-ai-xxxxx.vercel.app`
3. **把這個網址記下來**，這就是你的網站
4. 點進去試開，若看到 Hookvox 首頁就成功了

**若失敗**：點該次部署 → 看 **Logs** 或 **Building** 裡紅色錯誤，把錯誤訊息複製給我。

---

### 步驟 8：把「正確網址」填回環境變數（部署成功後）

1. 你步驟 7 拿到的網址可能是 `https://hookvox-ai-xxxxx.vercel.app`（中間有亂碼）
2. 到 Vercel → 你的專案 → 上方 **Settings** → 左側 **Environment Variables**
3. 找到 **NEXT_PUBLIC_APP_URL**、**ECPAY_NOTIFY_URL**、**ECPAY_RETURN_URL** 這三筆，點右邊 **Edit**
4. 把裡面的網址都改成**你實際拿到的網址**（例如 `https://hookvox-ai-xxxxx.vercel.app`），Notify 那筆結尾要是 `/api/ecpay/notify`，Return 那筆結尾要是 `/payment/success`
5. 存檔後，到 **Deployments** 分頁 → 最新一筆右邊 **⋯** → **Redeploy**，再等一次跑完

## 第三部分：綠界金流（讓大家能付錢）

### 步驟 9：你有沒有綠界帳號？

- **有、且已開通信用卡收款** → 做步驟 10
- **有、但還沒開通** → 登入 https://vendor.ecpay.com.tw（或綠界官網點廠商登入）→ 找「信用卡收款」或「申請服務」→ 依畫面填資料、上傳身分證 → 等 3～7 天審核 → 通過後做步驟 10
- **沒有** → 到 https://www.ecpay.com.tw 註冊會員 → 再申請特約商店／信用卡收款 → 同上

---

### 步驟 10：在綠界後台拿三組參數

1. 登入綠界**廠商後台**
2. 左側選單找 **「系統開發管理」** 或 **「API 介接」** 或 **「金鑰管理」**
3. 畫面上會看到：
   - **MerchantID**（特店編號）
   - **HashKey**
   - **HashIV**
4. 這三組你應該已經填在 .env.local 和 Vercel 了，**確認 Vercel 那三筆和這裡完全一致**（複製貼上最保險）

---

### 步驟 11：在綠界後台填「付款結果通知網址」

1. 同一個綠界後台頁面（或「信用卡設定」），找 **「付款結果通知網址」** 或 **ReturnURL** 或 **Notify URL**
2. 填：**你 Vercel 的網址 + `/api/ecpay/notify`**  
   例如：`https://hookvox-ai-xxxxx.vercel.app/api/ecpay/notify`
3. 一定要 **https**，網址不能多空格、結尾不要多斜線
4. 存檔

這樣使用者付完款，綠界才會通知你的網站更新訂閱狀態。

---

### 步驟 12：Supabase 設定（讓登入不會被擋）

1. 打開 https://supabase.com 並登入你的專案
2. 左側選 **Authentication** → **URL Configuration**
3. 在 **Redirect URLs** 那一區按 **Add URL**，填：`https://你的Vercel網址/**`  
   例如：`https://hookvox-ai-xxxxx.vercel.app/**`
4. 存檔
